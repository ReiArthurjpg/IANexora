# Localização do Botão de Login

O botão de login na tela inicial (`/`) está implementado no componente de navegação.

## Detalhes do Arquivo
- **Caminho:** `feature/guest/site/components/NexoraHero/ui/NexoraNavbar.tsx`
- **Componente:** `NexoraNavbar`

## Trecho de Código (Desktop)
```tsx
<a
  href="/guest/login"
  className="inline-flex items-center px-7 py-3 bg-white text-black text-[10px] font-black uppercase tracking-[0.25em] italic -skew-x-6 hover:bg-[#E11D48] hover:text-white transition-all duration-300 active:scale-95 shadow-lg"
>
  <span className="skew-x-6">Login</span>
</a>
```

## Trecho de Código (Mobile)
```tsx
<div className="mt-auto pt-8">
  <a
    href="/guest/login"
    className="w-full flex justify-center items-center py-4 bg-[#E11D48] text-white font-black uppercase tracking-widest italic -skew-x-6"
  >
    <span className="skew-x-6">Login</span>
  </a>
</div>
```

## Estilização
O botão possui um design premium com as seguintes características:
- **Inclinação:** `-skew-x-6` (Efeito de inclinação agressiva)
- **Cores:** Fundo branco com texto preto (Desktop), mudando para fundo vermelho (`#E11D48`) no hover.
- **Tipografia:** Fonte em negrito (`font-black`), todo em maiúsculas (`uppercase`), itálico (`italic`) e com espaçamento entre letras (`tracking-[0.25em]`).
